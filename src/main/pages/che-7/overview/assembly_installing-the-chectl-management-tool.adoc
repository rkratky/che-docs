= Installing the {prod-cli} management tool

:context: installing-the-{prod-cli}-management-tool

The {prod} command-line management tool is called `{prod-cli}`. It is used for operations on the {prod-short} server (start, stop, update, delete) and on workspaces (list, start, stop, inject) and to generate devfiles.
